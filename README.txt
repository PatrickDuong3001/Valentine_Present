This Android app is created in Java and Android Studio

App Content:
- This app is designed in a Quiz layout.
- The user has to choose the correct answer in order to proceed to the next questions.
- Whenver the user chooses the correct or incorrect answers, a corresponding dialog box will 
  always appear to indicate the correctness of the answer.
- If the user answers all the questions correctly, he/she will be led to the Gift Page. 

Features: 
- There's a security system (usernam + password verification) to protect the user's privacy
- Diverse color and background music system
- Friendly and eye-catchy UI design
- Dynamic embedded GIF images and dialog boxes
